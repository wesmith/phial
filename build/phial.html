About Phial¶

Phial (/ˈfī(ə)l/) is a toolbox for use with PyPhi. Its intent is to make it easy to do lots of IIT experiments. Our hope is that this toolbox will allow people that are not deeply imbedded in IIT research to run and share experiments.

Some features:

  • Generate TPM from node functions.

  • Generate all possible node functions.

  • Save and restore everything needed to reproduce an experiement.

  • Plot results.

  • Analyze network and state graphs.

  • Looping for doing lots of related experiments (e.g. calculate ϕ for all reachable states in a TPM)
