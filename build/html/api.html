API¶

Experiment¶

In addition to the API functionality, the experiment module provides a python script. Run it with: python3 experiment.py. See Running an experiment from the command line

NB: To use the graph DRAW methods, you must have graphviz installed. https://www.graphviz.org/

class experiment.Experiment(edges, title='', comment=None, funcs={}, states={}, net=None, saveDir=PosixPath('/home/pothiers/phial'), default_statesPerNode=2, default_func=<function MJ_func>)¶

Setup, run, analyze IIT experiments.

Nodes not given as keys to funcs dict default to ‘default_func’

Parameters

  • edges – connectivity edges; e.g. [(0,1), (1,2), (2,0)]

  • title – Label for experiment

  • comment – Additional reports associated with experiment

  • funcs – dict[nodeLabel] = func; func can be python function or a stripped name from library (node_functions.py) e.g. ‘XOR’.

  • states – dict[nodeLabel] = numStatesForNode

  • net – Net instance that defines network used for experiment

  • saveDir – directory to save experiment results into.

  • default_statesPerNode – default number of States per Node

  • default_func – default function (mechanism) for all nodes

Running an experiment from the command line¶

Run an IIT experiment

usage: experiment [-h] [--default_func DEFAULT_FUNC] [--SpN SPN]
                  [--outfile OUTFILE]
                  [--loglevel {CRTICAL,ERROR,WARNING,INFO,DEBUG}]
                  net

Positional Arguments¶

net

Net definition in JSON format containing at least “edges” key. Optional keys: “funcs”, “title”

Named Arguments¶

--default_func

Default function when not explicitly specified for a node.

Default: “XOR”

--SpN

Default number of states per node when not explicitly specified for a node.

Default: 2

--outfile

File to save experiment into

--loglevel

Possible choices: CRTICAL, ERROR, WARNING, INFO, DEBUG

Kind of diagnostic output

Default: “WARNING”

EXAMPLE: /home/pothiers/sandbox/phial/venv/bin/sphinx-build my_network.json”

toolbox.all_states(N, spn=2, backwards=False)¶

All combinations spn^N binary states in lexigraphical order. This is NOT the order used in most IIT papers. To get order for papers, set ‘backwards=True’. RETURN list of statestr in hex format. e.g. ‘01100’ spn:: States Per Node
